# Проєкт **Ruby Console Task Manager (CLI)**

**Формат**: консольний застосунок<br/>
**Мова**: чистий Ruby (БЕЗ Rails, БЕЗ Sinatra)<br/>
**Мета**: навчитися думати як бекенд-розробник, а не писати хаотичний код

## Головна ціль проєкту
**Навчитися:**
- писати структурований Ruby-код
- розбивати логіку на класи
- працювати з файлами
- робити просту архітектуру

## Функціонал (MVP - обов'язково)
1. **Задача (Task) <br/>**
   Кожна задача має:
    - id
    - title
    - description
    - status (todo / done)
    - created_at


2. **Можливості програми <br/>**
   У консолі користувач може:
    - Додати задачу
    - Показати всі задачі
    - Позначити задачу як виконану
    - Видалити задачу
    - Зберегти задачі у файл
    - Завантажити задачі з файлу
    - Вийти з програми

## Структура проєкту
**task_manager** <br/>
- main.rb
- task.rb
- task_manager.rb
- storage.rb
- tasks.json

## Технический стек
- Ruby
- JSON
